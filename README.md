Tema : Cine

Descripción : 

La idea es hacer un modelo de base de datos para un cine. Básicamente se va a poder ver qué películas están en cartel y en qué horarios, y el administrador va a ser el que pueda cargar o borrar películas, salas y funciones. El público común solo vería la info.

Modelo de datos :

Un género puede tener varias películas (acción, drama , comedia, etc.).
